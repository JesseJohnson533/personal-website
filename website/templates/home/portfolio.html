TFT Stat Aggregator and Tools

Date Completed: In Progress

This project is a TFT Stat Aggregator and Tools web application built with Next.js and TypeScript, offering a robust and scalable platform for Teamfight Tactics (TFT) players to connect their game accounts via Riot Games' external SSO. By consuming Riot's REST APIs, the app retrieves and stores personalized game data in a PostgreSQL relational database, enabling efficient data management and historical analysis. Users can access detailed statistics, match history, and performance insights on their account page, with tools to analyze gameplay, track progress, and optimize strategies. Combining the power of Next.js for a fast, responsive frontend and PostgreSQL for reliable data storage, this application delivers a seamless and type-safe experience for TFT enthusiasts.

Senior Project - Issac's Index

Date Completed: 12/2023

Issac's Index is a price and review aggregator Chrome extension and web application designed to revolutionize your online shopping experience. When browsing a product page, the extension displays prices from multiple retailers, ensuring you find the best deal, while the website curates user reviews from various sources, filtering out irrelevant or fake ones. It calculates an "Issac's Insight" score a Rotten Tomatoes like rating to help you make confident purchasing decisions based on collective shopper feedback. Built with Python (Flask, SQLAlchemy) for the backend, JavaScript, HTML, and CSS for the frontend, and PostgreSQL for data storage, the app scrapes the web using Playwright to gather real-time data. The Chrome extension is powered by Node.js and Bootstrap, while the entire system is cloud-deployed using Heroku, delivering a seamless and reliable shopping companion.

Corner! Restaurant Scheduling Mobile App

Date Completed: 08/2022

Corner! is a Android application built with Kotlin, designed to streamline scheduling, shift management, and financial tracking for restaurant staff. Following the MVC design pattern, the app includes 5 activities and screens. Key features include a RecyclerView for displaying shift notes, SQLite for structured storage, and CRUD operations for managing notes and shifts. Users can tap days to update tips and hours worked, long-tap to delete notes (with yes/no confirmation), or short-tap to edit them. Additional functionalities include a Weekly Financial box for tracking tips, a Number Spinner for post-shift updates, and the ability to switch between weeks seamlessly.
