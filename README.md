# personal-website

Personal website

Heroku Buildpacks:

-   heroku/python
-   heroku/nodejs

Local Development:
- Create a .env file in root folder
```
MAIL_USERNAME = ""
MAIL_PASSWORD = ""
MAIL_DEFAULT_SENDER = ""
```
- Open in devcontainer
- Run `pip install -r requirements.txt`
- Run `flask run --debug`
